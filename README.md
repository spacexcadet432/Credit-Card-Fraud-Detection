# Credit Card Fraud Detection using Machine Learning

## Problem Statement
Detect fraudulent credit card transactions using machine learning techniques.

## Dataset
Kaggle Credit Card Fraud Detection Dataset

## Models Used
- SVM
- Random Forest
- Decision Tree

## Methodology
- Data preprocessing (scaling, SMOTE)
- Model training
- Evaluation using ROC, Precision, Recall, F1-score

## Results
Random Forest achieved the best performance with highest ROC-AUC.

## Future Scope
- Real-time fraud detection
- Deep learning models
